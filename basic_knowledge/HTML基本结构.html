HTML头部 head

在HEAD中，可以插入title、style、meta、link、script、noscript、base。

title：定义html文档的标题。

base：定义了html文档中所有链接的默认目标地址，其格式和 a 标签一致。

meta：文档的描述、关键词、作者、字符集等。

HTML链接由标签 a 定义

这是一个链接。

链接内容可以是字、词、一组词、一幅图等。

HTML链接 — target属性

target属性用来指出被链接的文档在何处显示。
当其值设为"_blank"时，会在新窗口打开链接。
当其值设为"_top"时，链接会跳出框架。

HTML链接 — id属性

id属性相当于为元素设置一个书签，且可以通过链接（#id）进行引用。

HTML 元素

HTML 元素以开始标签（opening tag）起始，以结束标签（closing tag）终止；

大多数 HTML 元素可以嵌套使用；

HTML空元素

开始标签既标记开始，也标记结束。为了与XHTML以及其他未来版本HTML兼容，应该加上反斜线。

HTML 属性

属性以键值对的形式出现：属性名="值"

大多数元素支持的属性：class、id、style、title；
