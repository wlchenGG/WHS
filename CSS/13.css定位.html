CSS 定位相关样式[1]

定位相关的属性包括：

  • position、
  • top、
  • bottom、
  • left、
  • right、
  • z-index、
  • clip、
  • cursor、
  • overflow、
  • overflow-x、
  • overflow-y

元素定位属性 position

  • position: static; 默认值。没有特殊定位，就是一个普通的文档流内容，随着内容上下滚动而滚动。
  • position: fixed; 固定位置。相对于浏览器窗口固定位置。不会随着内容一起上下滚动。不占据文档流空间，因此会与文档流内容重叠。
  • position: relative; 相对位置。相对于这个元素正常位置的相对位置，与 top、bottom、left、right 一起使用来控制偏移距离。属于文档流，且仍然占据其正常位置的空间，而相对位置处不占用空间，可能会与文档流其他内容重叠。
  • position: absolute; 绝对位置。相对于最近的有定位父元素的位置，且不会随着内容一起上下滚动。不占据文档流空间，会与文档流内容重叠。
  • position: sticky; 粘性位置。行为取决于滚动位置，在 relative 和 absolute 之间切换。当页面滚动跨域某个阈值前，是 relative，跨域后就是 fixed，这个阈值由 top、bottom、left、right 来指定，且必须指定，否则就只是 relative 效果。

四边定位属性 top bottom left right

定义元素外边界相对于其包含元素对应边界的偏移。

同时需要指定定位类型。定位类型为 static 时，四边定位无效。定位类型为 relative 时，四边定位是相对于自身正常位置对应边界的偏移。定位类型为 absolute 时，四边定位是相对于其包含元素对应边界的偏移。
