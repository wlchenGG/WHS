id 选择器

这是一个特别的段落，定义了 id 为 para1，根据 id 单独设置了样式。
id 选择器以 #id {} 格式设置。 id 属性的值不能以数字开头。

class 选择器

可以为元素设置 class 属性，可以同时指定设置多个属性，只需以空格隔开，比如 class="color center"。

class筛选器的格式为

  • .center {} ：可以设置所有带 center 类属性值的元素格式
  • p.center {} ：可以单独设置带 center 类属性值的某一种元素格式

分组和嵌套

分组选择器

当多个元素具有相同的样式时，可以用分组选择器同时设置这些元素的样式：

只需将各个选择器以逗号间隔列出来即可。比如以下样式表将 h1、h2 和 p 三种元素都设置为绿色字体： 
      
        h1,h2,p {
          color: green;
        }

嵌套选择器

用于指定选择器内部某个或某些选择器的样式。

可以按嵌套顺序以空格隔开列出选择器即可：

.marked p{} ：为所有 class='marked' 元素里的 p 元素指定样式。

或者可以按嵌套顺序以.指定属于某类的选择器：

p.marked{} ：为所有 class='marked' 的 p 元素指定样式。
