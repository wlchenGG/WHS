CSS 表格相关样式[1]

表格边框 border

通过 border 属性设置，值为：粗细、线型、颜色，比如 border: 1px solid red;

如上设置，会使每一个边框线都是双线，即每个单元格都不共用边框线，如下所示：

A B
C D
可以通过 border-collapse: collapse 属性设置为单线：
A B
C D

宽高

通过 width 和 height 属性设置即可。

文字对齐

通过 text-align 属性设置水平对齐。取值 left、center、right

通过 vertical-align 属性设置垂直对齐。取值 top、middle、bottom

表格填充

文字与单元格边框之间的间距。通过 padding 属性控制。

表格颜色

通过 border 属性设置线颜色、background-color 设置背景颜色。
