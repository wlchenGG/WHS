CSS 列表相关样式[1]

列表项标记

通过属性 list-style-type 控制，可取值：

  • 无序列表
    • 'circle'、'square'
  • 有序列表
    • 'upper-roman'、'lower-alpha'、'cjk-ideographic'、'decimal'、'lower-greek' 等

但是这种方式不兼容所有浏览器，因此可以用以下设置背景图像的方式，手动设置列表项图像标记： 

  ul { 
       list-style-type: none;
       padding: 0px;
       margin: 0px;
    }
  ul li { 
       background-image: url('./images/li.png');
       background-repeat: no-repeat;
       background-position: 4px 8px; // 要注意控制图片大小、位置
       padding-left: 16px;
       
    }

链接分为四种状态：a:link、a:visiter、a:hover、a:active，都可以设置任何 CSS 样式（颜色、字体、背景等）。

设置时必须按照 link、visited、hover、active 顺序。
hover 必须在 :link 和 a:visited 之后定义才有效。
active 必须在 :hover 之后定义才有效。
