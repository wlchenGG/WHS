本 HTML 演示 CSS 背景属性

背景颜色 background-color

设置背景颜色
background-color: beige;

背景图像 background-image

设置背景图像
background-image: url("image.png")
默认平铺重复，铺满页面

图像重复模式 background-repeat: repeat-x|repeat-y|no-repeat

设置背景图像仅横向重复
background-repeat: repeat-x;

设置背景图像仅纵向重复
background-repeat: repeat-y;

设置背景图像不重复
background-repeat: no-repeat;

图像滚动 background-attachment: scroll|fixed|local

设置背景图像随页面滚动，是默认值
background-attachment: scroll;

图像位置 background-position

左上
background-position: left top;

上中
background-position: center top;
或 background-position: top;

右上
background-position: right top;

左中
background-position: left center;
或 background-position: left;

正中
background-position: center center;
或 background-position: center;

右中
background-position: right center;
或 background-position: right;

左下
background-position: left bottom;

下中
background-position: center bottom;
或 background-position: bottom;

右下
background-position: right bottom;

横百分比 纵百分比
background-position: 10% 80%;

横坐标 纵坐标
background-position: 100px 25px;

背景简写属性 background

按如下顺序依次给出（无需全部给出）：
  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

本例：
background: #af9fcf url('./images/planet.png') no-repeat right top;
