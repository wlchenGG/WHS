CSS 字体相关样式[1]

字体系列

字体系列分为 serif、sans-serif、monospace 三大类通用系列，每类具有相似的外观特征：

  • serif：字符末端带边角装饰。如 Times new roman、Georgia。
  • sans-serif：字符末不端带边角装饰。如 Arial、Verdana。
  • monospace：所有等宽字符具有相同宽度。如 Courier New、Lucida Console。

字体样式 font-style

字体样式分为 正常（normal）、斜体（italic）、倾斜文本（oblique）：

  • normal：正常字体，如“这是正常字体”
  • italic：斜体字体，如“这是斜体字体”。斜体字需要字体系列中具有相应的字体才能有效使用。
  • oblique：倾斜字体，如“这是强制倾斜文字”。当字体系列中没有相应斜体字体（即italic）时，用 oblique 强制倾斜文字。

字体大小 font-size

字体粗细 font-weight

通过属性 font-weight 属性设置字体的粗细程度，取值包括预定义关键词 normal、bold、bolder、lighter ，以及从 100 到 900 的整百数字，效果如下：

  • normal 正常字体
  • bold 粗体字符
  • bolder 更粗字体
  • lighter 更细字体
  • 100 最细字体
  • 400 等同 normal 字体
  • 700 等同 bold 字体

字体转变 font-variant

可以通过 font-variant 控制字体是否“小型大写”:

  • 正常字体 normal：The font variant。
  • 小型大写字体small-caps ：The font variant。其高矮取决于原字符的大小写。

简写字体属性 font

可以用 font 属性一起设置以上字体属性。可设置属性（按顺序）：
font-style font-variant font-weight font-size/line-height font-family

其他 font 属性值

  • caption：浏览器默认的 caption 控件字体。
  • icon：浏览器默认的 icon 标签字体。
  • menu：浏览器默认的下拉列表字体。
  • message-box：浏览器默认的对话框字体。
  • small-caption：浏览器默认的 caption 字体的小型版本。
  • status-bar 被窗口状态栏使用的字体：浏览器默认的窗口状态栏字体。
