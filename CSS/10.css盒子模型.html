CSS 盒子模型[1]

所有 HTML 元素都可以看成盒子。盒子模型包括：边距 margin、边框 border、填充 padding、内容

  • margin（外边距）：清除边框外的区域，是透明的。
  • border（边框）：内边距与外边距之间的边框。
  • padding（填充/内边距）：清除边框内的区域，是透明的。
  • content（内容）：显示内容主体。

通过 width 和 height 设置的只是内容大小。一个元素占据范围的总大小还包括内外边距、边框的大小，比如：

100 (width) + 20px (左+右填充) + 4px (左+右边框) + 20px (左+右边距) = 144px

CSS 边框 border

边框样式 border-style

属性值包括：dotted 点线、dashed 虚线、solid 实线、double 双边框、groove 3D沟槽、ridge 3D 脊边槽、inset 3D 嵌入边框、outset 3D 突出边框

包括四个边框，可以同时分别指定各边框样式：

  • 四个值：上、右、底、左
  • 三个值：上、左右、底
  • 二个值：上底、左右
  • 一个值：全部
此种方式也适用于 border-width 和 border-color。需要注意，如果要设置多个边框的多个属性值，需要按边框或者按属性分开设置，不能只用 border 设置。即上面同时为多边框指定不同值的方式只能适用于一个属性。

一个例子

边框线宽 border-width

属性值包括：thick、medium、thin，或者具体数值。

边框颜色 border-color

按颜色值设置即可。

简写属性 border

直接用 border 属性同时按顺序设置：线宽、线型（必须）、颜色。

边框圆角属性 border-radius

基本格式：水平半径s / 垂直半径s

水平半径和垂直半径均可以给出多个值，分别指定四个角的圆角半径：

  • 四个值：左上、右上、右下、左下
  • 三个值：左上、右上和左下、右下
  • 二个值：左上和右下、右上和左下
  • 一个值：全部

一个例子 border-radius: 8px 16px 32px 64px / 32px 64px 8px 16px;


一个例子 border-radius: 40% 10% 3% 30%/30% 10%;

CSS 轮廓 outline

轮廓 outline 是元素最外层的边缘框线，不会占据元素的 width 和 height。如上面的红色虚线框。形状会围绕 border 边框。

其类似 border 属性，具有 outline-style、outline-color、outline-width 属性，可以直接通过 outline 按顺序简写：颜色、线型、线宽。

CSS 外边距 margin

按顺序简写属性 margin：

  • 四个值：上、右、底、左
  • 三个值：上、左右、底
  • 二个值：上底、左右
  • 一个值：全部

margin: auto; 可将块居中。

值可为负。具体可参考 浅谈 margin 负值 - 柳兮

CSS 填充 padding

按顺序简写属性 padding：

  • 四个值：上、右、底、左
  • 三个值：上、左右、底
  • 二个值：上底、左右
  • 一个值：全部

值不可为负。
