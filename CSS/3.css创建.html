外部样式表

在 css 文件中通过选择器定义样式，然后再 html 头部用 link 标签引入 css 文件。

内部样式表

直接在 head 内以 style 标签的形式进行定义

内联样式

直接在要要设定样式的标签内，以 style 属性的形式进行定义。

多重样式以及优先级

同一个属性可能在不同样式表里同时被定义，此时按照 更具体的优先，
优先级顺序为：内联样式 > 内部样式表 > 外部样式表 > 浏览器默认样式

但是要注意，如果在head内，引入外部样式表的 link 语句在 style 标签后面，那么此时，外部样式表 > 内部样式表

CSS 中选择器的优先级

内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器 > 通配选择器
