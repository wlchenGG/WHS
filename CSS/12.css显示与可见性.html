CSS 显示相关样式[1]

属性 display 指定元素如何显示。

属性 visibility 指定元素是否隐藏。

隐藏元素

display 和 visibility 都可以用来隐藏元素，但效果不同：

  • display: none; - 隐藏元素的同时，在页面上也不会占据位置空间。
  • visibility: hidden; - 隐藏元素的同时，在页面上仍然占据位置空间。

不隐藏：
外面隐藏外面

display: none;
外面隐藏外面

visibility: hidden;
外面隐藏外面

控制表格中某行列隐藏可以用 display: none; 或者 visibility: collapse; 这样能保证表格外边框自动调整。如果用 visibility: hidden; 则隐藏的单元格会空白一块。

改变块级或内联显示

块元素：占用全部宽度，前后都有换行符。比如 h1、p、div

内联元素：占用必要宽度，不自动换行。比如 span、a、img

可以通过 display 改变元素以块级或者内联形式显示：

  • display: inline; 内联显示。比如可以将列表项显示在同一行。
  • display: block; 块级显示。
