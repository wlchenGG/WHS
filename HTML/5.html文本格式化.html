基本字体格式化

文本格式化通过标签来设置。

这是粗体，用 b 标签

这是着重文本，用 strong 标签

这是斜体，用 i 标签

这是加重语气，用 em 标签

这是加大字体，用 big 标签

这是缩小字体，用 small 标签

这是上标字体，用 sup 标签

这是下标字体，用 sub 标签

预格式化

      这是一个预格式化文本段落，
      将按照源代码格式
      原样 显示出来。用 pre 标签。
      看起来字号比正常段落字号小。

计算机输出，常用于显示代码

计算机输出，Computer output，用 code 标签
键盘输入，keyborad input，用 kbd 标签
打字机文本，typing text，用 tt 标签
计算机代码样本，code sample，用 samp 标签
计算机变量，conputer variable，用 var 标签
定义项目，用 dfn 标签

地址

显示地址，用 address 标签

Written by Jon Doe.

Example.com

USA

缩写和首字母缩写

etc.，用 abbr 标签，titile 属性指明全称。鼠标移到词上面时，会显示全称。

WWW，用 acronym 标签，titile 属性指明全称。鼠标移到词上面时，会显示全称。

引用

短引用：鲁迅：横眉冷对千夫指，俯首甘为孺子牛。 ——用的是标签 q，会自动将内容用双引号括起来。

引用块：用 blockquote 标签，会自动两端缩进，例子：

在 HTML 4.01 中，blockquote 标签定义一段长引用。在 HTML5 中，blockquote 标签定义摘自另一个源的块引用，用 cite 属性指明网页来源。如需把页面作为 XHTML 进行验证，那么 blockquote 元素必须包含块级元素。

引证：用 cite 标签，会自动斜体显示。用来定义作品的标题：

星星之火可以燎原，毛泽东
