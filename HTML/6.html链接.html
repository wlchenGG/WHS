链接标签的属性

target 属性

target 属性指明将在哪里显示这个链接页面，默认在当前标签页直接跳转，值 "_blank" 表示在浏览器新标签页打开；"_top" 可以跳出当前的框架；"_self" 在同一个窗口打开新的页面；"_parent" 在父窗口中打开新的页面。

这个链接从新标签页打开我的主页。

id 属性

id 给出这个链接内容在当前 html 文档中的唯一标识，可以作为书签来进行引用。
